# Backend Final Exam

[Spanish version](README.es.md)

Academic project developed for the backend final exam at Tecsup. The work is divided into two Spring Boot microservices focused on CRUD operations, PostgreSQL persistence, validation, and deployment.

## Microservices

- `ms-productos`: product management, update, and logical deletion.
- `ms-pedidos`: order management, total calculation, and status update.

## Technical Focus

- Layered architecture: controller, service, repository, dto, entity, and exception.
- REST APIs with Spring Boot.
- Persistence with Spring Data JPA and PostgreSQL.
- Input validation with Jakarta Validation.
- Containerization with Docker.
- Deployment preparation for Render.

## Status

Completed academic project used as Tecsup evidence.
