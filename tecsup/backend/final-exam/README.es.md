# Examen Final Backend

[English version](README.md)

Proyecto academico desarrollado para el examen final de backend en Tecsup. El trabajo esta dividido en dos microservicios Spring Boot orientados a operaciones CRUD, persistencia con PostgreSQL, validaciones y despliegue.

## Microservicios

- `ms-productos`: gestion de productos, actualizacion y eliminacion logica.
- `ms-pedidos`: gestion de pedidos, calculo de total y actualizacion de estado.

## Enfoque Tecnico

- Arquitectura por capas: controller, service, repository, dto, entity y exception.
- APIs REST con Spring Boot.
- Persistencia con Spring Data JPA y PostgreSQL.
- Validacion de entrada con Jakarta Validation.
- Contenerizacion con Docker.
- Preparacion para despliegue en Render.

## Estado

Proyecto academico finalizado como evidencia de Tecsup.
