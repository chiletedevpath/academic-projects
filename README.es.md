# Proyectos Academicos

[English version](README.md)

Proyectos academicos de desarrollo de software de UTP y Tecsup, organizados por institucion, curso y tema.

## Proposito

Este repositorio agrupa tareas, examenes, laboratorios y proyectos de curso sin mezclarlos con practica personal o proyectos de portafolio.

## Estructura

- `utp`: proyectos y trabajos de UTP.
- `tecsup`: proyectos y trabajos de Tecsup.

## Criterio

Los proyectos de este repositorio son evidencia academica. Un proyecto puede convertirse luego en proyecto de portafolio si se pule, se documenta y aporta valor mas alla del curso.
