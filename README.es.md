# Academic Projects

[English version](README.md)

Repositorio de proyectos academicos de Chilete DevPath. Agrupa entregas, examenes, retos y proyectos formales desarrollados en UTP y Tecsup.

## Proposito

Mantener evidencia academica ordenada sin mezclarla con ejercicios de aprendizaje (`learning-labs`) ni con proyectos personales de portafolio. Cada carpeta debe conservar el contexto de institucion, curso y objetivo.

## Estructura

- `utp`: proyectos y trabajos desarrollados en UTP.
- `tecsup`: proyectos y trabajos desarrollados en Tecsup.

## Contenido Actual

- `tecsup/backend/final-exam`: microservicios `ms-pedidos` y `ms-productos`.
- `tecsup/backend/sunat-consulta`: backend para consulta y evidencia relacionada.
- `tecsup/frontend`: retos y proyectos frontend de clase.
- `utp/databases/database-i/ferreteria-soto-db`: proyecto final de Base de Datos I.
- `utp/design-patterns/ferreteria-sys-patrones`: proyecto academico de patrones de diseno.
- `utp/web-development/la-luchona`: proyecto web academico.

## Criterio

Los proyectos de este repositorio son evidencia academica. Un proyecto puede convertirse luego en proyecto de portafolio si se pule, se documenta, se despliega y aporta valor mas alla del curso.
