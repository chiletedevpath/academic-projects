# Ferreteria Soto DB

[Spanish version](README.es.md)

Final Database I project developed for UTP. It models the main operations of a hardware store: users, customers, product classification, products, sales, auditing, and pricing policies.

## Contents

- `00-create-database.sql`: database creation.
- `01-usuarios.sql`: users, roles, states, and auditing.
- `02-clientes.sql`: customer management.
- `03-clasificacion-productos.sql`: categories and classifications.
- `04-productos.sql`: products and inventory.
- `05-ventas-detalle-auditoria.sql`: sales, details, and auditing.
- `06-politicas-precios.sql`: pricing rules and policies.
- `codigo-fuente-bd-ferreteria-soto.sql`: integrated project script.
- `reset-inventario-ventas.sql`: helper script for data reset.
