# Ferreteria Soto DB

[English version](README.md)

Proyecto final de Base de Datos I desarrollado para UTP. Modela operaciones principales de una ferreteria: usuarios, clientes, clasificacion de productos, productos, ventas, auditoria y politicas de precios.

## Contenido

- `00-create-database.sql`: creacion de la base de datos.
- `01-usuarios.sql`: usuarios, roles, estados y auditoria.
- `02-clientes.sql`: gestion de clientes.
- `03-clasificacion-productos.sql`: categorias y clasificaciones.
- `04-productos.sql`: productos e inventario.
- `05-ventas-detalle-auditoria.sql`: ventas, detalle y auditoria.
- `06-politicas-precios.sql`: reglas y politicas de precios.
- `codigo-fuente-bd-ferreteria-soto.sql`: script integrado del proyecto.
- `reset-inventario-ventas.sql`: script auxiliar para reinicio de datos.
