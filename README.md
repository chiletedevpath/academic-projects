# Academic Projects

[Spanish version](README.es.md)

Academic software development projects from UTP and Tecsup, organized by institution, course, and topic.

## Purpose

This repository groups academic assignments, exams, labs, and course projects without mixing them with personal practice or portfolio projects.

## Structure

- `utp`: projects and coursework from UTP.
- `tecsup`: projects and coursework from Tecsup.

## Criteria

Projects in this repository are academic evidence. A project can later become a portfolio project if it is polished, documented, and useful beyond the course context.

## Status

In progress.
