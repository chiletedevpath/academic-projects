# Academic Projects

[Spanish version](README.es.md)

Academic projects repository for Chilete DevPath. It groups formal assignments, exams, challenges, and course projects developed at UTP and Tecsup.

## Purpose

This repository keeps academic evidence organized without mixing it with learning exercises (`learning-labs`) or personal portfolio projects. Each folder should preserve its institution, course, and objective context.

## Structure

- `utp`: projects and coursework developed at UTP.
- `tecsup`: projects and coursework developed at Tecsup.

## Current Contents

- `tecsup/backend/final-exam`: `ms-pedidos` and `ms-productos` microservices.
- `tecsup/backend/sunat-consulta`: backend for consultation and related evidence.
- `tecsup/frontend`: class challenges and frontend projects.
- `utp/databases/database-i/ferreteria-soto-db`: Database I final project.
- `utp/design-patterns/ferreteria-sys-patrones`: academic design patterns project.
- `utp/web-development/la-luchona`: academic web project.

## Criteria

Projects in this repository are academic evidence. A project can later become a portfolio project if it is polished, documented, deployed, and useful beyond the course context.
